============================================================
一、已完成功能 ✅
============================================================

[核心拦截]
  ✅ 系统 WebView 强制使用，不内嵌第三方内核
  ✅ 黑白名单双模式：白名单模式仅允许列表内网址，
     黑名单模式禁止列表内网址
  ✅ 通配符域名匹配：支持 *.school.edu.cn 格式
  ✅ 所有请求拦截：shouldOverrideUrlLoading + shouldInterceptRequest
  ✅ 拦截页面显示中文提示

[UI 功能]
  ✅ 顶部线性进度条，页面加载时显示
  ✅ 地址栏：显示当前 URL + 支持输入导航 + 策略校验
  ✅ 快捷地址：横向滚动标签组 + 下拉弹窗列表
  ✅ 锁图标：HTTPS 时显示绿色锁

[远程策略拉取]
  ✅ HTTP/HTTPS 拉取（OkHttp 4.12）
  ✅ 可选跳过 SSL 证书验证（内网自签证书场景）
  ✅ SMB 协议拉取（jcifs-ng 2.1.10）
  ✅ SMB 密码 AES-256-GCM 加密存储
  ✅ 远程策略 JSON 缓存到本地

[策略优先级]
  ✅ 远程策略 > 缓存远程策略 > 本地 .txt 文件 > 本地加密策略
  ✅ 白名单模式下 whitelist 为空 → 策略无效 → 显示"等待配置"
  ✅ 黑名单模式下 blacklist 为空 → 不拦截任何网址（有效）
  ✅ 本地 .txt 文件自动读取：
     - 白名单：policy_data/true_website.txt
     - 黑名单：policy_data/notrue_website.txt
     - 格式：每行一个域名，# 注释行，支持通配符
  ✅ 强制本地模式开关

[安全设计]
  ✅ 管理密码 BCrypt 哈希 (cost=12)
  ✅ EncryptedSharedPreferences 存储敏感数据
  ✅ AES-256-GCM 加密本地策略（Android Keystore 硬件保护）
  ✅ 密码错误 3 次锁定 30 分钟
  ✅ 恢复码错误 5 次锁定
  ✅ ProGuard 代码混淆
  ✅ 禁止文件访问 / 多窗口 / 下载
  ✅ debuggable = false

[管理面板]
  ✅ 长按 3 秒 → 密码验证 → 管理面板
  ✅ 首页地址设置
  ✅ 快捷地址增删
  ✅ 远程源配置（HTTP/HTTPS / SMB、跳过 SSL、强制本地）
  ✅ 修改管理密码
  ✅ 首次设置向导：设置密码 + 生成恢复种子

[文档]
  ✅ README.md（功能特性、技术栈、编译指南、JSON 格式、安全设计）

四、已知限制 & 注意事项 ⚡
============================================================

  ⚡ minSdk = 26 (Android 8.0)，不支持更低版本
  ⚡ 自适应图标 API 26+，低版本设备需额外 png 图标
     （当前未提供 png 回退，低版本使用默认 Android 图标）
  ⚡ SMB 依赖 jcifs-ng，需 SMB v2/v3 服务器
  ⚡ 策略 JSON 未签名，存在中间人篡改风险
  ⚡ 管理面板未做 Session 超时，打开后一直有效直到返回
  ⚡ WebView 无法拦截 WebSocket 连接
  ⚡ 地址栏在无策略时可输入任意 URL → 但实际加载会被拦截
